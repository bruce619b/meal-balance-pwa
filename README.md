# 刷餐計算機 PWA v5

## v3 更新
- 快速選項改成自助餐 90 / 110 / 125 / 140
- 7-11 標價輸入從 1 欄改成 5 欄商品明細
- 5 欄會自動加總，再乘以 85 折
- 在 7-11 標價總和下方新增「7-11 標價折扣後總和」
- 保留每日自動重置與離線使用

## v4 新功能
- 主卡改為「7-11 標價還可買」
- 商品輸入後即時扣除剩餘可買標價
- 超過安全上限時顯示 0 並提示超出金額

## v5 新功能
- 主卡明確改為「7-11 折後剩餘可買標價」
- 先計算已輸入商品的 85 折扣款
- 從剩餘餐費扣除折後扣款
- 再把剩餘餐費除以折扣率，反推還可買的商品標價
- 提示列同步顯示：已輸入標價、折後扣款、剩餘餐費、還可買標價
